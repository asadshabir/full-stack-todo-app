# FastAPI (Backend) Skill

Expert in Python FastAPI. Use async/await for all database and I/O operations. Follow RESTful conventions. Use Pydantic schemas for request validation and response modeling in the /backend directory.

## Core Competencies

- **Async Programming**: Proficient in using async/await for database and I/O operations
- **RESTful APIs**: Following REST conventions for endpoint design
- **Pydantic Schemas**: Using Pydantic for request validation and response modeling
- **FastAPI Framework**: Leveraging FastAPI's features for efficient backend development
- **Database Operations**: Implementing async database operations with proper error handling

## Best Practices

1. Always use async/await for database operations
2. Implement proper request validation with Pydantic models
3. Follow RESTful conventions for API endpoints
4. Use dependency injection for shared resources
5. Implement proper error handling and response models
6. Include comprehensive API documentation
