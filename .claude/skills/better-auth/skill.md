# Better Auth (Security) Skill

Expert in Better Auth integration. Manage JWT token issuance on the frontend and verification on the FastAPI backend using a shared BETTER_AUTH_SECRET. Ensure strict user isolation for all Todo data.

## Core Competencies

- **Better Auth Integration**: Implementing Better Auth for authentication
- **JWT Management**: Handling JWT token issuance and verification
- **Frontend Integration**: Managing authentication on the frontend
- **Backend Verification**: Verifying tokens on the FastAPI backend
- **Security**: Ensuring secure authentication flows
- **User Isolation**: Strict separation of user data
- **Shared Secrets**: Proper management of BETTER_AUTH_SECRET

## Best Practices

1. Secure JWT token issuance and verification
2. Proper handling of BETTER_AUTH_SECRET
3. Ensure strict user isolation for all data
4. Implement secure authentication flows
5. Validate tokens on every protected endpoint
6. Follow security best practices for authentication
7. Proper error handling for authentication failures
