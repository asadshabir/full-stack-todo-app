# OpenAI Agents (AI Logic) Skill

Expert in OpenAI Agents SDK. Build multi-agent workflows that can use MCP tools to perform CRUD operations. Focus on maintaining conversation state in the database while keeping the server stateless.

## Core Competencies

- **OpenAI Agents SDK**: Proficient in using the OpenAI Agents SDK
- **Multi-Agent Workflows**: Building complex workflows with multiple agents
- **MCP Integration**: Integrating MCP tools for CRUD operations
- **Conversation State**: Managing conversation state in the database
- **Stateless Architecture**: Keeping the server stateless while managing state externally
- **CRUD Operations**: Performing create, read, update, delete operations through agents
- **Workflow Design**: Designing efficient multi-agent workflows

## Best Practices

1. Maintain conversation state in the database
2. Keep the server stateless for scalability
3. Integrate MCP tools effectively for data operations
4. Design efficient multi-agent communication patterns
5. Implement proper error handling for agent workflows
6. Follow OpenAI Agents SDK best practices
7. Optimize for performance and reliability
