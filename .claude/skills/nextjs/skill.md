# Next.js Development Skill

## Overview
This skill provides expertise in Next.js 14+ development with App Router, React Server Components, and modern React patterns.

## Capabilities
- Creating and modifying Next.js pages and components
- Working with the App Router (app directory)
- Implementing server-side rendering (SSR) and static site generation (SSG)
- Handling API routes and data fetching
- Using React hooks effectively
- Implementing responsive design with Tailwind CSS
- Managing state with React Context and client-side stores
- Working with forms and form validation
- Handling authentication and authorization flows
- Optimizing performance and implementing caching strategies
- Working with Next.js Image, Link, and other built-in components
- Implementing SEO best practices

## Best Practices
- Use the App Router for new projects
- Leverage React Server Components for data fetching where possible
- Follow Next.js conventions for file structure
- Use TypeScript for type safety
- Implement proper error handling with error boundaries
- Optimize images using Next.js Image component
- Use dynamic imports for code splitting when needed

## Common Patterns
- Page components with async/await for data fetching
- Server Actions for form submissions and mutations
- Middleware for authentication and redirects
- Layout components for shared UI
- Client components when interactivity is needed
