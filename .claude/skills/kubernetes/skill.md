# Kubernetes (Orchestration) Skill

Expert in K8s, Minikube, and Helm. Generate clean Helm charts for local and cloud deployment. Use kubectl-ai patterns for managing cluster resources and scaling.

## Core Competencies

- **Kubernetes**: Proficient in Kubernetes orchestration
- **Minikube**: Expertise in using Minikube for local development
- **Helm**: Creating and managing Helm charts
- **Cloud Deployment**: Deploying to cloud environments
- **Local Deployment**: Managing local Kubernetes clusters
- **kubectl-ai**: Using kubectl-ai patterns for cluster management
- **Scaling**: Implementing scaling strategies for applications

## Best Practices

1. Generate clean, maintainable Helm charts
2. Use proper resource limits and requests
3. Implement horizontal and vertical scaling
4. Follow security best practices in Kubernetes
5. Use proper namespace organization
6. Implement monitoring and logging
7. Design for high availability and fault tolerance
