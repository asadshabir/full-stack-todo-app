# MCP (Model Context Protocol) Skill

Expert in Official MCP SDK. Design and implement stateless MCP servers that expose Python functions as tools for AI agents. Follow strict JSON-RPC standards for tool communication.

## Core Competencies

- **MCP SDK**: Proficient in using the Official MCP SDK
- **Stateless Servers**: Designing stateless MCP servers
- **Tool Exposure**: Exposing Python functions as tools for AI agents
- **JSON-RPC Standards**: Following strict JSON-RPC standards for communication
- **AI Integration**: Integrating with AI agents through MCP protocol
- **Server Implementation**: Implementing MCP servers with proper error handling

## Best Practices

1. Design stateless servers to ensure scalability
2. Follow strict JSON-RPC standards for communication
3. Properly expose Python functions as tools
4. Implement comprehensive error handling
5. Ensure secure communication protocols
6. Follow MCP specification guidelines
7. Test tool communication thoroughly
